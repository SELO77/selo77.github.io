Docker를 이용한 지속적 배포 환경 구축

2016-07-19
  • CD
  • DevOps
  • Docker

Docker Compose 란 멀티 컨테이너의 실행을 위한 툴입니다. 한번의 실행으로 미리 정의된 모든 서비스를 생성하고 시작시킬수 있습니다.
클러스터

Celery 란 Python으로 작성된 비동기 작업 큐입니다. 작업(메시지)을 브로커를 통해 전달하면 하나 이상의 워커가 이를 처리하게 됩니다. 예를 들면 사용자에게 이메일 발송, SMS 발송, 게시물 등록 등 비동기 처리가 필요한 작업을 Celery에게 위임하고 처리할 수 있습니다.
